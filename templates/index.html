Ready for learning something new more easily?

The Flashcards Learning Method are a great tool of any student that needs to memorize new information. Instead of forcing all the information in your head all at once, breaking up the material in small note card sized bits can prove more effective both for memorizing and retaining information. And this website gives you all of that power, especially when it comes to learning a new language. Some suggested steps to use the website most effectively:

The first step is to create new flashcards. With /decks you can first create a folder. Then dive into this folder and make your own cards. Here you can edit, delete or flip cards and folders. /list will give you the same features but with a tabular template to have an overview of all the new words you have created.

Secondly click on /learn to switch to learn mode. You can select a folder that you have created. Then start learning the cards in this folder. There are several features that may be useful to you: again to go back to the first card of the deck, shuffle to shuffle the deck, normal to learn the cards in order, and out to spin again and select another folder.

Finally, you can visit /list, a different template than /decks, to see all your cards in one table. And below the table there is a text area. You can write a text using all the new words to learn them more effectively. You can also practice new words by playing memory games at /decks. Choose a random card, guess its meaning, then turn the card over and see your answer.
